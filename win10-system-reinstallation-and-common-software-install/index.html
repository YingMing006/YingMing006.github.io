系统重装

win10 家庭版

下载windows10 官网，点击立即下载，通过该工具制作U盘启动盘。（最简单、最方便）

win10 专业版

准备

需要 win10 专业版的镜像，下载途径：

  • 正版软件管理和服务平台（大学正版软件平台，推荐）
  • ITELLYOU （提供原版软件）
  • IT之家-WIN10系统下载

U盘启动盘制作软件：Rufus、UltrISO

U盘启动盘制作

Rufus 开箱即用 操作简单

UltrISO 使用步骤：

  • “文件” - “打开”，打开ISO镜像文件
  • “启动” - “写入硬盘镜像”
  • 写入方式：“USB-HDD+”，“写入”，等待写入完成
